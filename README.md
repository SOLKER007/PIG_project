Pig Dice Game 
A Python-based implementation of the classic Pig Dice Game for 2-4 players. Take turns rolling a die, accumulate points and decide whether to roll or hold to reach 100 points and win!
Features
•	Supports 2-4 players.
•	Interactive turn-based gameplay with roll/hold choices.
•	Automatic score tracking and winner declaration.
•	Fully console-based; no external dependencies.
How to Play
1.	Run the script:
2.	python pig_dice_game.py
3.	Enter the number of players.
4.	On your turn, choose to roll or hold. Rolling a 1 ends your turn with 0 points.
5.	First player to 100 points wins!
Technologies
•	Python 3.x
•	Random module for dice rolls
